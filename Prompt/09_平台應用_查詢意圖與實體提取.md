## 9.1 查詢意圖識別（`identify_entities_in_query`）

- **用途**：Graph RAG 主流程中，`db_service.search_entities()` 無結果時的 fallback，從問題中識別資安術語以再次比對圖譜節點。
- **對應 API**：`POST /api/chat`（fallback）
- **輸出格式**：JSON 陣列（資安術語清單）

### 系統提示詞

```text
從問題中識別資安術語，回傳 JSON 陣列
```

### 使用者提示詞

```text
{query}
```

### 變數說明

- `{query}`：使用者輸入之問題。

---

## 9.2 文字實體關係提取（`extract_entities_from_text`）

- **用途**：從任意文字提取實體與關係（輔助用途）。
- **對應 API**：內部函式，目前未掛載 API 路由
- **輸出格式**：JSON 陣列，格式 `[{"source": "...", "relationship": "...", "target": "..."}]`

### 系統提示詞

```text
你是資安知識圖譜專家。從文字中提取實體和關係，輸出 JSON 格式:
[{"source": "實體A", "relationship": "關係類型", "target": "實體B"}]
```

### 使用者提示詞

```text
{text}
```

### 變數說明

- `{text}`：待提取之輸入文字。
