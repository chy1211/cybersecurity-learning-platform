## 8.1 單主題出題（`generate_quiz`）

- **對應 API**：`POST /api/quiz/generate`
- **輸出格式**：JSON 陣列，每題含 `id`、`question`、`options`（4 選項）、`correctAnswer`（0–3 整數）

### 系統提示詞

```text
你是一個專業的資安考題出題老師。
```

### 使用者提示詞

```text
        你是資安教育專家。請根據以下知識圖譜內容，針對「{entity}」這個主題，生成 10 題單選題。
        
        參考知識圖譜內容：
        {context}
        
        題目風格要求：
        1. 類似專業證照考試（如初級資訊安全工程師）。
        2. 包含情境題與觀念題。
        3. 難度適中。
        4. 每個選項必須有具體內容，不能只有 "A", "B" 等代號。
        
        輸出格式要求：
        請直接輸出 JSON 格式，不要包含 markdown 標記。
        格式如下：
        [
            {
                "id": "q1",
                "question": "題目內容...",
                "options": ["選項A內容", "選項B內容", "選項C內容", "選項D內容"],
                "correctAnswer": 0
            },
            ...
        ]
        注意：correctAnswer 必須是 0, 1, 2, 或 3 的整數，分別代表 A, B, C, D。
```

> 程式碼中 JSON 範例之大括號以雙重大括號 `{{ }}` 跳脫（Python `str.format`）；上方已還原為實際送入 LLM 之單層大括號樣貌。

### 變數說明

- `{entity}`：出題主題（知識圖譜節點名稱）。
- `{context}`：該主題之知識圖譜參考內容。

## 8.2 多主題批量出題（`generate_batch_quiz`）

- **對應 API**：內部函式，目前未掛載 API 路由
- **輸出格式**：JSON 陣列，每題額外含 `entity_name` 欄位

### 系統提示詞

```text
你是一個專業的資安考題出題老師。
```

### 使用者提示詞

```text
        你是資安教育專家。請針對以下多個主題，每個主題生成 2 題單選題。
        
        主題列表與參考內容：
        {context_str}
        
        題目風格要求：
        1. 類似專業證照考試。
        2. 每個主題出 2 題。
        3. 難度適中。
        4. 每個選項必須有具體內容。
        
        輸出格式要求：
        請直接輸出 JSON 格式，不要包含 markdown 標記。
        格式如下：
        [
            {
                "id": "q1",
                "entity_name": "主題名稱",
                "question": "題目內容...",
                "options": ["選項A", "選項B", "選項C", "選項D"],
                "correctAnswer": 0
            },
            ...
        ]
```

`{context_str}` 由迴圈組裝，每項格式為：

```text
主題：{entity}
參考內容：{context}
-------------------
```

### 變數說明

- `{context_str}`：多主題之參考內容串接字串（每項含主題名稱與該主題知識圖譜內容）。
